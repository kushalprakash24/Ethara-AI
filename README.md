# Ethara-AI: Enterprise Desk & Seating Management System

An interactive, real-time enterprise desk allocation and workspace monitoring solution. Built with a modern technical stack, this application bridges manual layout coordination with automated onboarding pipelines, featuring a live graphical floor plan grid layout and intelligent search telemetry.

---

## 🚀 Technical Stack

*   **Frontend UI:** Next.js / React.js, Tailwind CSS (Utility-first styling Engine)
*   **Backend Server:** FastAPI (Asynchronous Python Web Framework), Uvicorn Engine
*   **Database Engine & ORM:** PostgreSQL Core, SQLAlchemy Object Relational Mapper

---

## ✨ Features

*   **Interactive Grid Map:** Live visual mapping of enterprise assets categorized into real-time operational states (`vacant`, `occupied`).
*   **Dynamic Onboarding Pipeline:** Seamless joiner onboarding workflow via automated backend first-available seat resolution or explicit targeted override configurations.
*   **Manual Override & Swap Management:** Explicit binding adjustments linking target workspace coordinates to distinct entity identifiers (`employee_id`, `seat_id`, `project_id`) while safely tracking and flushing prior allocation paths automatically.
*   **Enterprise Search Indexing:** Live, asynchronous global search functionality providing dynamic record queries with real-time payload filtering.
